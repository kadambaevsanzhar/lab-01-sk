Task 2 — Tech roles mapping (Spotify)

1\. Key roles



Frontend Engineer — web UI



Mobile Engineer — iOS / Android apps



Backend Engineer — APIs and business logic



DevOps / SRE — infrastructure and reliability



QA Engineer — testing and quality



Security Engineer — security and access control



Data Engineer — analytics pipelines



ML Engineer — recommendations and ML models



2\. Components and ownership



Client apps (Web/Mobile/Desktop) — Frontend, Mobile



API Gateway / Load Balancer — SRE, Backend



Auth / Identity — Backend, Security



Catalog \& Library services — Backend



Playback service — Backend, SRE



Cache \& Messaging — Backend, SRE, Data



Analytics \& ML systems — Data, ML



Monitoring \& Security tools — SRE, Security



3\. Operational responsibility



Service availability — SRE



Data issues — Backend



Security incidents — Security



Content delivery issues — SRE



4\. Team collaboration



Frontend/Mobile ↔ Backend — API contracts



Backend ↔ Data/ML — events and data flows



SRE ↔ all teams — deployments and incidents



QA ↔ developers — release testing
